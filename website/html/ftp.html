FTP information

The LEGO pages on the World-Wide Web is the definitive source for LEGO-related material. However, LEGO is also available by ftp transfer from ftp.dcs.ed.ac.uk in pub/lego. You can get the system as follows: 

ftp ftp.dcs.ed.ac.uk
Name: anonymous
Password: enter your e-mail address
cd pub/lego
get README
Then, please read the file README.
